## Overview

JOIN is a Kanban-based task management application designed to help teams organize tasks, manage workflows, and track progress efficiently.

---

## Features

- Kanban board with multiple workflow stages  
- Create, edit, and delete tasks and subtasks  
- Assign tasks to contacts  
- Due date management  
- User registration and login  
- Contact management  
- Fully responsive design (desktop, tablet, mobile)

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend / Database:** Firebase Realtime Database

---

## Local Setup

1. Clone the repository  
2. Replace the Firebase Realtime Database URL with your own configuration  
3. Open `index.html` in your browser
